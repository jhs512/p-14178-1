# 05 - 글 수정·삭제 (작성자 권한)

Status: ready-for-agent

## Parent

`.scratch/board/PRD.md`

## What to build

작성자가 본인 글을 수정·삭제하는 경로 전체. 소유권 검증을 서비스 레벨에서 수행해 타인의 글 수정·삭제를 거부한다.

- 수정: 제목·마크다운 본문 갱신.
- 삭제: 글 제거(연관 댓글/추천 처리 포함).
- 소유권: 작성자 본인만 허용, 그 외 거부.

## Acceptance criteria

- [ ] 작성자가 본인 글을 수정하면 제목·본문이 갱신된다.
- [ ] 작성자가 본인 글을 삭제하면 목록/상세에서 사라진다.
- [ ] 타인이 수정·삭제를 시도하면 거부된다(권한 오류).
- [ ] 비로그인 접근 시 로그인으로 유도된다.
