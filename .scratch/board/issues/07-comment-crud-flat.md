# 07 - 댓글 작성·조회·수정·하드삭제 (평면)

Status: ready-for-agent

## Parent

`.scratch/board/PRD.md`

## What to build

회원이 글에 평면 댓글을 달고, 글 상세에서 댓글 목록을 보고, 작성자가 본인 댓글을 수정·삭제하는 경로 전체. `PostComment` 엔티티(`content` 평문, 소속 `Post` N:1, 작성자 `Member` N:1)와 서비스 CRUD. 이 슬라이스에서는 평면(최상위) 댓글만 다루며, 삭제는 단순 하드 삭제다. (대댓글/소프트 삭제는 08)

- 댓글 작성·수정·삭제는 로그인 필요, 조회는 공개.
- 소유권: 작성자 본인만 수정·삭제.

## Acceptance criteria

- [ ] 로그인 회원이 글에 댓글을 작성하면 글 상세에 표시된다.
- [ ] 글 상세에 해당 글의 댓글 목록이 보인다(비로그인도 조회).
- [ ] 작성자가 본인 댓글을 수정하면 반영된다.
- [ ] 작성자가 본인 댓글을 삭제하면 완전히 제거된다(하드 삭제).
- [ ] 타인이 수정·삭제를 시도하면 거부된다.
