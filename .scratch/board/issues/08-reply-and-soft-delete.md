# 08 - 대댓글(1단계) + 소프트 삭제 분기

Status: ready-for-agent

## Parent

`.scratch/board/PRD.md`

## What to build

댓글에 1단계 대댓글을 달고, 댓글 삭제 시 대댓글 유무로 소프트/하드 삭제를 분기하는 경로 전체. `PostComment`에 자기참조 `parent`(nullable)와 `isDeleted` 플래그를 추가한다.

- 대댓글은 최상위 댓글만 부모로 가질 수 있다(대댓글에 답글 불가 — 2단계 금지).
- 삭제 분기: 대댓글이 있는 부모 댓글 삭제 시 행을 지우지 않고 `isDeleted=true`로 두고 내용을 "삭제된 댓글입니다"로 가린다(대댓글 보존). 대댓글이 없으면 하드 삭제.
- 상세 화면에서 대댓글이 부모 아래 들여쓰기로 표시되고, 삭제된 부모는 플레이스홀더로 표시된다.

## Acceptance criteria

- [ ] 회원이 최상위 댓글에 대댓글을 달면 부모 아래에 표시된다.
- [ ] 대댓글에 다시 답글을 달 수 없다(2단계 금지).
- [ ] 대댓글이 없는 댓글을 삭제하면 행이 제거된다(하드).
- [ ] 대댓글이 있는 댓글을 삭제하면 `isDeleted=true`로 남고 "삭제된 댓글입니다"로 표시되며 대댓글은 보존된다(소프트).
- [ ] 통합 테스트(인메모리 H2): 대댓글 무→하드 삭제, 대댓글 유→소프트 삭제+대댓글 보존.
