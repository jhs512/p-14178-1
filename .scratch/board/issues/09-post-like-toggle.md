# 09 - 글 추천 토글 (PostLike)

Status: ready-for-agent

## Parent

`.scratch/board/PRD.md`

## What to build

회원이 글을 추천(토글)하고 추천수가 집계되는 경로 전체. `PostLike` 엔티티(`Member` N:1 + `Post` N:1, 복합 유니크 제약)와 토글 서비스.

- 토글: 추천이 없으면 생성, 있으면 취소(삭제).
- 한 회원은 한 글에 1회만 추천(유니크 제약으로 중복 방지).
- 본인 글도 추천 가능.
- 글 목록/상세의 "추천수"가 PostLike 개수로 표시된다.

## Acceptance criteria

- [ ] 로그인 회원이 글 추천을 누르면 추천수가 1 증가한다.
- [ ] 이미 추천한 글을 다시 누르면 추천이 취소되어 추천수가 감소한다.
- [ ] 같은 글을 두 번 추천할 수 없다(유니크 제약).
- [ ] 본인 글도 추천된다.
- [ ] 비로그인 추천 시 로그인으로 유도된다.
