# 트리아지 라벨

스킬들은 5개의 표준 트리아지 역할로 말합니다. 이 파일은 그 역할을 이 저장소 이슈 트래커에서 실제 사용하는 라벨 문자열에 매핑합니다.

| mattpocock/skills 라벨 | 우리 트래커 라벨   | 의미                                  |
| ---------------------- | ----------------- | ------------------------------------- |
| `needs-triage`         | `needs-triage`    | 관리자가 이 이슈를 평가해야 함          |
| `needs-info`           | `needs-info`      | 보고자의 추가 정보 대기 중             |
| `ready-for-agent`      | `ready-for-agent` | 완전히 명세됨, AFK 에이전트 착수 준비완료 |
| `ready-for-human`      | `ready-for-human` | 사람의 구현이 필요함                   |
| `wontfix`              | `wontfix`         | 처리하지 않음                          |

스킬이 역할을 언급하면(예: "AFK 준비완료 트리아지 라벨을 적용하라"), 이 표에서 해당하는 라벨 문자열을 사용합니다.

실제 사용하는 어휘에 맞게 오른쪽 열을 수정하세요.
